배열의 고차함수
    고차함수는 함수를 인자로 전달받거나 함수를 결과로 반환하는 함수

    * callback function(콜백함수)
        부모 함수의 인자에 자식 함수가 전달되어 부모 함수가 자식 함수에게 매개변수를 
        전달하고 실행하는 함수를 말한다.

        function callback(func){
            let a = 0;
            func(a)
        }
        callback((num)=> console.log(num))

        callback() -> 실행문
                -> func() -> 실행문

        ... (test.js 참고)

        ---------------------------------------------------------------------------
    map
        배열을 순회하며 각 요소에 대하여 인자로 주어진
        콜백함수의 반환값으로 *새로운 배열* 을 생성

    filter
        경우에 따라 if문 대체 가능, 배열을 순회(모든 요소를 탐방)하며
        각 요소에 대하여 인자로 주어진 콜백함수의 결과가 true인 요소만 추출하여 
        *새로운 배열* 을 만든다.
        => 내가 원하는 요소로 이루어진 배열

    find
        배열을 순회하며 인자로 주어진 콜백함수를 실행하고 그 결과가 참인 것 중
        첫 번째인 *요소* 를 반환한다.
        => 내가 원하는 요소

    findIndex
        find와 기능은 같지만 첫 번째 요소의 인덱스를 반환 
        => 내가 원하는 요소의 인덱스 

    reduce
        누적값을 구할 때 사용
        요소들의 계산된 누적값과 현재 요소 변수를 return

    every
        해당 배열의 요소가 모두 특정 조건을 만족하는지 boolean 형태

    some
        해당 배열의 요소 중 일부가 특정 조건을 만족하는지 boolean 형태로 반환

    sort
        정렬을 위해 사용, 콜백함수의 현재값과 다음값을 비교하여
        비교식에 따라 순서를 정렬

        arr.sort((a, b) => a-b)
