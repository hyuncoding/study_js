
addEventListener
    자바스크립트에서 이벤트를 등록하는 가장 권장되는 방식
    하나의 요소에 여러 개의 이벤트를 등록할 수 있다는 장점

    ex)
        요소 || window.addEventListener("이벤트명", 콜백함수, options)
        콜백함수는 이벤트가 실행되었을 때 실제 동작하게 될 기능
        options 캡쳐링의 유무 default 값은 false


  자주 사용하는 이벤트명
  -- document --
    click: 요소를 클릭했을 때
    mouseover: 요소에 마우스를 올렸을 때
    mouseleave: 요소에 마우스가 떠났을 때
    mousedown: 요소에서 마우스를 누르고 있는 중
    mouseup: 요소에서 마우스를 누르고 있는 것을 뗐을 때
    mousewheel: 요소에서 마우스가 움직일 때마다

  -- input --
    focus: 활성화되었을 때 
    blur: 활성화에서 비활성화되었을 때
    keypress: 키를 누른 순간
    keyup: 키를 누르고 있다가 뗀 순간
    keydown: 키를 누른 상태
    change(oninput): input의 value가 바뀌었을 때
    submit: form 태그와 같이 사용, form이 제출되었을 때

  -- window, body --
    load: 웹브라우저가 열렸을 때
    resize: 웹 브라우저의 사이즈가 조정되었을 때 (최소화, 최대화)
    scroll: 스크롤바의 움직임이 생겼을 때
    unload: 웹브라우저가 종료되었을 때

------------------------------------------------------------------

캡쳐링, 버블링

  버블링
      한 요소에 이벤트가 발생되면, 이어서 부모의 이벤트 콜백함수 역시 동작
      하위 요소의 이벤트가 부모 요소로 전파

  캡쳐링
      부모 요소를 클릭했는데 자식 요소가 클릭되는 현상을 캡쳐링
